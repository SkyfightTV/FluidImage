
# FluidImage

Display your image with a fluid collision system.

## Run

Clone the project

```bash
  git clone https://github.com/SkyfightTV/FluidImage.git
```

Go to the project directory

```bash
  cd FluidImage
```

Create binary file

```bash
  mvn package 
```

Launch the program

```bash
  java -jar ./FluidImage [image_path]
```
